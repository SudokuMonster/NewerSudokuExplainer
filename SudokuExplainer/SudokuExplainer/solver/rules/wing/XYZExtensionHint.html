XYZ-Wing Extension

The four cells {0}, {1}, {2} and {3} form an XYZ-Wing extension pattern on the value {4}. Regardless of the correct value of the cell {0} ({4}, {5} or {6}), one of the three cells {1}, {2} or {3} will have the value {4} as its only remaining possible value. It follows that among the cell {1}, {2} and {3} must contain the value {4}.

Other occurances of the value {4} can therefore be removed from any cells sharing a row, column or block with all {1}, {2} and {3}.
