Continuous Nice Loop

The cells {0} form a Continuous Nice Loop: there are exactly two ways of placing the values in these cells, forming two different possible configurations. Some values appear in a row, column or block regardless of the chosen configuration. Because one of the two configurations must be correct, other occurances of these values can be removed from these rows, columns or blocks.
